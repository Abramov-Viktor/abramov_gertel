Инварианты: открыто главное окно запущенного приложения, доступна строка поиска действий и переход к панели быстрого доступа. Основное меню включает 10 пунктов, среди которых доступ к основным пользовательским сценариям, настройка приложения, шифрование данных. Отдельно выделен пункт быстрого монтирования образа (внизу) с использованием параметров по умолчанию.

Первый пользовательский сценарий: монтирование образа.
Действующее лицо – пользователь (во всех пользовательских сценариях).
Предусловие: наличие образа диска одного из поддерживаемых программой форматов, сама программа установлена.
Основной сценарий:
1. Запустить программу;
2. Выбрать пункт меню «Монтировать» (рисунок 1);
3. Выбрать пункт «Образ» (рисунок 2);
4. Указать путь к образу и необходимые параметры (альтернативно – выбрать файл, пользуясь поиском по проводнику, нажав на «три точки»), указать тип привода и метку тома, в которую будет произведено монтирование (рисунки 3, 4)
5. Нажать кнопку «Монтировать» (результат на рисунке 5);
6. Открыть смонтированный диск, нажав на иконку образа с рисунка 5 (результат на рисунке 6).
Альтернативный сценарий:
1.	Выбрать пункт меню «Быстрое монтирование»;
2.	Альтернативный вариант 4 пункта основного сценария;
3.	Остальные действия согласно 5, 6 пунктам.
Постусловие сценариев: изображено на рисунках 5, 6. Диск готов к просмотру.

Второй пользовательский сценарий: Запись диска с данными.
Предусловие: программа установлена.
Основной сценарий:
1.	Запустить программу.
2.	Перейти во вкладку «Записать» (рисунок 7).
3.	Выбрать «Диск с данными» или «Audio CD» в зависимости от формата файлов (рисунок 8).
4.	Выбрать привод и файлы для записи, установить параметры.
5.	Нажать кнопку «Старт».
Альтернативный сценарий:
1.	Повторить 1 и 2 пункты основного сценария.
2.	Выбрать «Пишущий привод» (рисунок 8).
3.	Выбрать CD или DVD в зависимости от потребностей, указать место сохранения, выбрать формат (рисунок 9).
4.	Нажать кнопку «Старт».
5.	Повторить 3 пункт основного сценария.
6.	Выбрать файлы для записи в окне проводника, нажав на «плюсик», задать параметры (рисунок 10).
7.	Нажать кнопку «Старт».
Постусловие сценариев: диск готов, данные записаны.

Третий пользовательский сценарий: создание загрузочного образа на USB.
Предусловие: имеется образ, USB носитель.
1.	Запустить программу.
2.	Выбрать пункт меню «Создать».
3.	Выбрать пункт «Загрузочный USB» (рисунок 11).
4.	Выбрать образ с помощью всплывающего окна проводника и носитель, установить параметры (рисунок 12).
5.	Нажать кнопку «Старт».
Постусловие: создан загрузочный USB носитель (рисунок 13).

Четвёртый пользовательский сценарий: очистка диска.
Предусловие: имеется диск с записанной информацией, программа установлена.
1.	Запустить программу.
2.	Выбрать пункт «Записать».
3.	Выбрать пункт «Стереть диск» (рисунок 8).
4.	Задать параметры (рисунок 14).
5.	Нажать кнопку «Старт».
Постусловие: диск очищен.

Пятый пользовательский сценарий: дублировать диск.
Предусловие: имеется диск с записанной информацией и пустой диск, программа установлена.
1.	Запустить программу.
2.	Выбрать пункт «Записать».
3.	Выбрать пункт «Копирование дисков» (рисунок 8).
4.	Задать параметры (рисунок 15).
5.	Нажать кнопку «Старт».
Постусловие: на двух дисках записана одинаковая информация.

Шестой пользовательский сценарий: запись образа на диск.
Предусловие: имеется пустой диск, образ для записи.
1.	Запустить программу.
2.	Выбрать пункт «Записать».
3.	Выбрать пункт «Образ» (рисунок 8).
4.	Выбрать образ, привод, параметры (рисунок 16).
5.	Нажать кнопку «Старт».
Постусловие: образ записан на диск (рисунки 17, 28).

Седьмой пользовательский сценарий: изменение образа.
Постусловие: программа установлена, имеется образ для редактирования.
1.	Запустить программу;
2.	Выбрать пункт «Редактировать»;
3.	Выбрать любой пункт в зависимости от цели (рисунок 19);
4.	Произвести нужные операции (рисунок 20);
5.	Нажать кнопку «Далее»;
6.	Выбрать параметры сохранения (рисунок 21);
7.	Нажать кнопку «Старт».
Постусловие: содержимое или тип образа изменён.
